## 优先级建议（更新版）

按投入产出比排序：

**已完成（✅）**：多城市横划切换、多城市概览面板、自然语言天气摘要、逐小时湿度图

**低成本高价值（优先做）**：
1. **#1 点击 7 天预报跳当日逐小时** — 用户强预期的标准交互
2. **#4 农历 / 节气** — 中国用户强需求，纯前端零 API 成本
3. **#8 一句话天气文案** — 彩云标志性体验，模板实现
4. **#5 AQI 健康建议文案** — 纯模板，数据已有
5. **#9 分钟级降水文字描述** — `summary` 字段已在接口返回，直接展示
6. **#3 月出 / 月落时间** — 接口已有字段，组件补充展示
7. **#2 降水量 mm** / **#6 云量** — hourly 页数据补充
8. **#13 首页内联穿衣建议** — 减少跳页，数据已在首页批次中

**中等成本**：
9. **#10 30 天降水概率图** — 新 Canvas 图表
10. **#11 生活指数等级可视化** — 样式改动
11. **#12 城市时区提示** — 小功能，面向特定用户群

**长期目标**：天气雷达 / 降水地图（成本最高，体验提升最大）
