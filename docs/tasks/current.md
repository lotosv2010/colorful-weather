## 新增页面

### P0 ★★★★★ 极高价值 / 接口全现成

#### #1 `pages/calendar` 天气月历
- **功能**：月历视图展示当月每天天气图标 + 最高/最低温，比 weather30 折线图更直观
- **数据**：未来用 `weather30d`，过去用 `historicalWeather` 拼接完整当月数据
- **入口**：首页 daily 组件"30天预报 ›" → weather30 页"月历"tab（第三个 tab，内容区内嵌切换）
- **状态**：`[x] 已完成（2026-07-21）`

#### #2 `pages/compare` 城市对比
- **功能**：横向对比 2 个城市的实时天气 + 7 天温度趋势 + 空气质量，旅行 / 异地用户刚需
- **数据**：`now` + `sevenDay` + `air`（均已有）
- **入口**：city-overview 面板右上角「对比」按钮，选中 2 个城市后跳转
- **状态**：`[x] 已完成（2026-07-21）`

#### #3 `pages/astronomy` 天文详情
- **功能**：独立展示日出 / 日落时刻、太阳高度角全天曲线（Canvas）、月相动画、月出 / 月落。当前天文信息埋在首页底部，独立成页可大屏展示
- **数据**：`sun` + `moon` + `solarElevationAngle`（均已有）
- **入口**：首页 astronomy 组件点击跳转
- **状态**：`[x] 已完成（2026-07-22）`

---

### P1 ★★★★ 高价值 / 需一定开发量

#### #4 `pages/trip` 出行规划
- **功能**：选择目的地城市 + 出行日期，展示目标城市未来天气 + 生活指数综合评分，给出「晴好 / 带伞 / 勿出行」结论
- **数据**：`sevenDay` + `indices3d` + `air`（均已有）；城市选择复用 `city-search` 组件
- **入口**：设置页或首页 bottom-sheet 新增入口卡片
- **状态**：`[x] 已完成（2026-07-21）`

#### #5 `pages/share` 精美天气分享卡
- **功能**：Canvas 绘制天气卡片（城市名 + 温度 + 天气图标 + 空气质量 + 日期），预览后保存图片分享；目前仅支持文字分享
- **数据**：复用现有 data，无需新接口
- **入口**：首页 full-list 「分享天气」入口卡（出行规划卡下方）
- **状态**：`[x] 已完成（2026-07-22）`

#### #6 `pages/air-station` 空气质量监测站
- **功能**：展示附近监测站的实时 PM2.5 / PM10 / NO₂ / O₃ 数值与站点名称；`airStation` API 已封装但无入口
- **数据**：先从 `airHourly` 响应取 `stations[]`，再逐站调 `airStation`
- **入口**：空气质量页底部新增「周边监测站」区块，点击展开或跳转
- **状态**：`[x] 已完成（2026-07-22）`

### P2 ★★★ 中等价值

#### #7 深色模式
- **功能**：全局 CSS 变量适配系统 `darkMode`；已有 `--theme` 变量基础，扩展一套暗色 token 即可
- **影响范围**：所有页面与组件的 `.wxss`
- **状态**：`[~] 不实现` — 微信小程序原生框架对暗色模式的接口支持存在限制，暂不实现

#### #8 桌面小组件 (Widget)
- **功能**：微信小程序桌面 widget，展示当前城市温度 + 天气图标，无需打开即可看天气
- **数据**：`now`（已有）；偏好复用 `prefs.js`
- **状态**：`[~] 不实现` — 微信小程序桌面 widget API 成熟度有限，且需额外审核，暂不实现

#### #9 首页节气 / 节日高亮卡片
- **功能**：`lunar.js` 已有节气算法，在首页 bottom-sheet 新增一个节气 / 传统节日高亮卡片
- **实现**：新增 `components/solar-term/` 组件；`utils/lunar.js` 扩展 `getSolarTermCard()`；插入 full-list `weather-life` 之后
- **状态**：`[x] 已完成（2026-07-22）`

#### #10 地图 POI 天气增强
- **功能**：点击地图 POI 时，利用 `poiLookup` 返回 POI 名称展示在 tips 中；目前仅显示逆地理编码的行政区名
- **状态**：`[x] 已完成（2026-07-22）`
